# Basic Syntax

All Markdown applications support these elements.

# Heading

# H1

# H1

# H2

# H2

# H3

### H3

# Bold

bold text

**bold text**

# Italic

italicized text

*italicized text*

# Blockquote

blockquote

> blockquote

# Ordered List

  1. First item
  2. Second item
  3. Third item
1. First item
2. Second item
3. Third item

# Unordered List

  • First item
  • Second item
  • Third item
- First item
- Second item
- Third item

# Code

code

`code`

# Horizontal Rule

---

# Link

title

[title](https://www.example.com)

# Image

![alt text](image.jpg)
